My Projects

Fossil - Water Potability Prediction App (2024)

Fossil is a Streamlit-based Machine Learning application designed to predict the potability of water using various chemical and physical parameters such as pH, sulfate concentration, and organic carbon. Built with an MLPClassifier model, this tool helps users assess whether water is safe to drink. Developed as part of the final project for the Google Developer Student Clubs Open Class (GDGoC) at Telkom University.

By Fauzan Ahsanudin Alfikri - April 2025

COPPA Risk Prediction - FIND IT UGM 2025

A machine learning model built with XGBoost to predict whether a mobile application is at risk of violating the Children's Online Privacy Protection Act (COPPA). This project was developed for the Data Analytics Competition FIND IT 2025, hosted by Universitas Gadjah Mada. It includes preprocessing, feature engineering, class balancing, and model evaluation using AUC metrics.

By Fauzan Ahsanudin Alfikri - April 2025
